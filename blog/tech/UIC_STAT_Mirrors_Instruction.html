R & [CRAN] Mirrors Manual

UIC-STAT now maintains its own CRAN mirror, http://mirrors.uicstat.com/CRAN/ and and it syncs with the main CRAN mirror in Austria(http://cran.r-project.org) once per day.

The Intranet CRAN mirrors is for the convenience of the stuffs and students to download various required libs in the internal network environment, without the problems of disconnection or downloading. In addition, you can use this image to deploy the R environment on your PC and server faster.

For Windows Users

The Latest R's Version is 3.5.3 (May 25th, 2019）， You can install the 32-bit version for your old computer (windows 7 and the systems before windows 7). However, if you do not need the 32-bit version, please do not select 32-bit software when you access to the download interface.

Click this link: http://mirrors.uicstat.com/CRAN/bin/windows/base/ and follow the instruction, you can get the latest R or the specific version you want.

Instead of using the intranet-mirror, you can use the URL provide from TsingHua University or University of Science and Technology of China

  • https://mirrors.tuna.tsinghua.edu.cn/CRAN/bin/windows/base/
  • http://mirrors.ustc.edu.cn/CRAN/bin/windows/base/ 。

For Mac or Linux Users

For the Mac users, you just need to change the link into http://http://mirrors.uicstat.com/CRAN/bin/macosx/ and follow the instruction, then you can install the latest R

For the Linux Users, you can follow the instruction CRAN has written in Choose Your Linux Version(only for debian, redhat, suse, and ubuntu). Also, you may need to sign the gpg-key first before you update the source.list

[CRAN] - HOWTO

You can use our mirror，I can promise that in the intranet, this CRAN-mirrors can be the fastest mirror.

Just type this command in your R-GUI:

xxxxxxxxxx
options(repos = c(CRAN = "http://mirrors.uicstat.com/CRAN/"))

ATTENTION, if you want to use the STAT's CRAN-mirrors, please consider the scene outside the intranet. Hence, you may change the command into:

Using UIC-[CTAN] Mirrors install packages

By modifying the image source, you can download the various packages and documents required by TeX User more quickly. The following is a description of the use of tlmgr:

The CTAN (The Comprehensive TeX Archive Network) mirror source can be changed using the TeXLive Manager tlmgr.

type this command in your terminal 

xxxxxxxxxx
tlmgr option repository http://mirrors.uicstat.com/CTAN/systems/texlive/tlnet

You can permanently change the mirror source.

If you only change it temporarily, you can use the following command:

xxxxxxxxxx
tlmgr update --all --repository http://mirrors.uicstat.com/CTAN/texlive/tlnet

———— However, if you cannot access the UIC's intranet, you can use the Tsinghua's link to download it：：

xxxxxxxxxx
tlmgr update --all --repository https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/tlnet

 

CRAN 镜像使用帮助

内网 CRAN 镜像是为了方便同学老师在内网环境中下载各种需要的 lib，而不会出现断连接、无法正常下载等问题。另外，也可以通过这个镜像，更快在个人 PC，服务器上部署 R 环境。

部署 R

Windows 用户

目前 R 的版本是 3.5.3，值得庆祝的是 R 在 2019 年依旧支持 32 bit 系统，这就证明了 windows 7 以及之前的系统依旧可以下载最新版的 R 。 你可以选择在 https://cran.r-project.org/bin/windows/base/R-3.5.3-win.exe 下载，也可以选择使用 UIC CRAN 镜像：http://mirrors.uicstat.com/CRAN/bin/windows/base/R-3.5.3-win.exee 进行下载。

当然，你也可以使用友校的连接：

  • https://mirrors.tuna.tsinghua.edu.cn/CRAN/bin/windows/base/R-3.5.3-win.exe
  • http://mirrors.ustc.edu.cn/CRAN/bin/windows/base/R-3.5.3-win.exe

当然，64bit 系统的同学或者老师可以选择不下载 32bit 的版本，这样可以节省大约 200MB 的空间。

Mac 和 Linux 用户

在 macOS 上下载 R 和在 Windows 上大同小异: 只需要将连接改成：http:///CRAN/bin/macosx/R-3.5.3.pkg

而 Linux 用户，请根据镜像站的指示自行完成下载、安装。

修改镜像源 方法一 CRAN (The Comprehensive R Archive Network) 镜像配置文件之一是 .Rprofile Linux 下 位于(~/.Rprofile)

在文末添加如下语句：

{R} options(repos = "http://mirrors.uicstat.com/CRAN/") 打开 R 后即可使用该 CRAN 镜像源安装 R 软件包

方法二 打开 R GUI，然后输入：

{R} options(repos = "http://mirrors.uicstat.com/CRAN/") 建议 在选择在个人笔记本上使用 UIC CRAN 镜像时，建议考虑外网使用场景（因为 UIC CRAN 镜像暂时属于内网镜像）；这是你将上述语句修改为：

{R} options(repos = c("http://mirrors.uicstat.com/CRAN/", "https://mirrors.tsinghua.edu.cn/CRAN/", "https://mirrors.ustc.edu.cn/CRAN/")) 可以看出，这里添加了两个友校（清华大学和中国科技大学）的镜像链接。这样，可以保证在外网访问时，正常的使用 CRAN 。

使用 UIC CTAN 镜像源进行包下载

通过修改镜像源，可以更加快速的下载 TeX User 所需各类宏包的和文档。下面是 tlmgr的介绍是使用方法说明：

CTAN(The Comprehensive TeX Archive Network) 镜像源可以使用 TeXLive 管理器 tlmgr 更改。

在命令行中执行：

{shell} tlmgr option repository http://mirrors.uicstat.com/CTAN/systems/texlive/tlnet 即可永久更改镜像源。

如果只是临时更改，可以用如下命令：

{shell} tlmgr update --all --repository http://mirrors.uicstat.com/CTAN/texlive/tlnet 其中，update --all 指令可根据需要修改

注意！在非 UIC 区域，请将修改自己的镜像源，避免出现无法正常下载的现象

这里提供友校的连接：

{shell} tlmgr update --all --repository https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/tlnet
